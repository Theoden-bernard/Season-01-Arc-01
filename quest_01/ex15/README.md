# DESCRIPTION

Create a my_isdigit function.

Reproduce the behavior of isdigit() function. It returns 1 if the character sent as argument is a digit (0 to 9). It returns 0 otherwise.

```c
int my_isdigit(char param_1)
{

}
```

Input: "a"
Output: 
Return Value: 0
```
## Example 01

Return Value: 0
```
## Example 02

Input: "0"
Output: 
Return Value: 1
