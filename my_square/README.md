# DESCRIPTION

Create a program which displays a beautiful square.

my_square(5,3) should display:
```bash
$>./a.out 5 3
o---o
|   |
o---o
$>
```
my_square(5, 1) should display:
```bash
$>./a.out 5 1
o---o
$>
```
my_square(1, 1) should display:
```bash
$>./a.out 1 1
o
$>
```
my_square(1, 5) should display:
```bash
$>./a.out 1 5
o
|
|
|
o
$>
```
my_square(4, 4) should display:
```bash
$>./a.out 4 4
o--o
|  |
|  |
o--o
$>
```
